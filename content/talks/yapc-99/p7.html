What to do with a database handle?

  • It's just a reference! (again)
  • Pass it to subroutines

Passing a handle to a subroutine:

 my $dbh = DBI->connect(...);
 ...
 my @ordered_users = find_users( $dbh );
 ...
 sub ordered_users {
  my $dbh = shift;
  ...
 }
 
 or

 my @ordered_users = find_users( dbh => $dbh );
 ...
 sub ordered_users {
  my %p = @_;
  $p{dbh}->do( $sql_statement );
  ...
 }
