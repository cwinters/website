SQL Basics

  • SQL == Structured Query Language
  • ANSI standard language for accessing relational database systems
  • Flavors of SQL "beyond" ANSI: PL/SQL (Oracle), Transact-SQL (Sybase, Microsoft SQL Server)

SQL examples:

Data definition:

 CREATE TABLE widget_info (
  widget_id int not null auto_increment,
  widget_name varchar(50),
  widget_ref varchar(15),
  widget_inventor varchar(200)
 );
 
Data manipulation:

 SELECT widget_name, widget_ref 
   FROM widget_info
  WHERE widget_name like '%chainsaw%'

Note: the widget_id definition would in Sybase/MS SQL Server be:

 widget_id numeric(12,0) identity,
