What to do with a database handle?

  • Use it in a HAS-A object relationship:

Code:

  sub full_name {
   my $self = shift;
   $self->{dbh}->do( $sql_statement );
   ...
 }

Note that if you have lots of classes using the same module, it might be best to create one class that holds all top-level objects and create a method call to return a database handle when asked for:

 my $rv = $self->dbh()->do( $sql );
