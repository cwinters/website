Binding parameters: SELECT example

Something else to watch out for:

 my $inventor = 'Bunc';
 my $sql = qq/
   SELECT widget_id, widget_ref, widget_inventor
     FROM widget_info
    WHERE widget_inventor LIKE ?
 /;
 my ( $sth );
 eval {
  $sth->prepare( $sql );
  $sth->execute( $inventor );
 };

This will not find all the records with 'Bunc', just the one that equals 'Bunc'. If you want to use wildcards, set the wildcards in the variable before binding:

 my $inventor = 'Bunc%';
