Attributes, Attributes

  • Last element in the DBI connect call
  • Sets attributes for the database handle
  • Attributes can be overriden by statement handles (children) or later modified
  • Special metadata attributes (more later)

Some descriptions:

 (default in parens)

 RaiseError   - If positive, raises a die
                when encountering an error (0)
 PrintError   - Same, but prints the error to STDERR rather 
                than throwing things around (1)
 AutoCommit   - Commits every statement before going on to 
                next one (more later)

Example of later modification:

 $dbh->{AutoCommit} = 1;
