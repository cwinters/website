Simple Statements

  • The DBI support for simple (e.g., non-select) statements is via the do method.
  • Return value will be the number of rows affected
  • Truth and no rows affected: 0E0

Example of DELETE:

 my $sql = qq/
   DELETE FROM widget_info
    WHERE widget_ref = '7D8915-XL'
 /;
 my ( $rv );
 eval {
  $rv = $dbh->do( $sql );
 };
 if ( $@ or ! $rv ) {
   my $err = $@ || $DBI::errstr;
   db_error( error => $err, sql => $sql );
 }
 print "$rv rows affected by delete\n";
