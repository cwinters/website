Using Your New Plugin

  • If not under 'Template::Plugin' specify the namespace using 'PLUGIN_BASE'
  • Example: given plugin 'My::Server::Framework::MyPlugin'
my $template = Template->new(
                   PLUGIN_BASE => 'My::Server::Framework'
               );
$template->process( \*DATA, {} );
__DATA__
[% USE MyPlugin %]
