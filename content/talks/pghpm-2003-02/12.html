Using Your New Plugin - Trick for skipping USE

  • Instead of having TT create the plugin for you DIY
  • Works best with stateless plugins
  • Good for frameworks -- extra template functionality is just part of the environment
my $template = Template->new(
                   PLUGINS => { MyPlugin => 'My::Server::Framework::Plugin' } );
my $plugin_instance = $template->context->plugin( 'MyPlugin' );
$template->process( \*DATA, { MyPlugin => $plugin_instance } );
__DATA__
Look ma, no USE!
Result: [% MyPlugin.result %]
