Using Your New Plugin - Associate a different name

  • You can dissociate a plugin from the package name
  • Example: given plugin 'My::Server::Framework::Plugin'
my $template = Template->new(
                   PLUGINS => { MyPlugin => 'My::Server::Framework::Plugin' }
               );
$template->process( \*DATA, {} );
__DATA__
[% USE MyPlugin %]
