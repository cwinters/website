Using a Plugin -- Stateful

  • Template::Plugin::CGI
  • Provides functionality of CGI.pm
  • Pseudo stateless use -- looks like class calls to me:
[% USE CGI %]
You entered: <br>
[% FOREACH name = CGI.params %]
   [% name %] = [% CGI.param( name ) %]<br>
[% END %]
  • Can also be more obviously stateful assigning to a variable:
[% USE q = CGI %]
You entered: <br>
[% FOREACH name = q.params %]
   [% name %] = [% q.param( name ) %]<br>
[% END %]
