Create an Action: Content handler

What is a content handler?

It performs actions

These actions are the heart of an application

Manipulate objects and information and pass them off for display

Content handler format:

Each action is a Perl class

Each individual task within an action is a method within the class

Example Action: User

Task: list (lists active users)

Task: show (displays a single user for reading/editing)

Task: edit (edits information for a single user)

Task: remove (removes a single user)

Example Action: ContactSearch

Task: search (searches a contact database)

Task: export (exports search results to interchange format)

Task: email (sends email to search results)
