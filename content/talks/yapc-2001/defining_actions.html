Action table

So how we match the Perl class with a URL?

Action table

Package can define one or more entries for the action table

Each entry maps a top-level URL to a Perl class, a component, or something that generates content

Can contain metadata about the content

What does an entry look like?

Perl configuration file format (Data::Dumper)

$action = {
     'user'    => {
         'class'    => 'OpenInteract::Handler::User',
         'security' => 'no',
     },
     'newuser' => {
         'class'    => 'OpenInteract::Handler::NewUser',
         'error'    => [ 'OpenInteract::Error::User' ],
         'security' => 'no',
     },
};
