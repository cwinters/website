What is OpenInteract?

From the gospels of the README:

OpenInteract is an extensible application server using Apache and mod_perl built for developers but also to be manageable via the web browser.

What goes into it:

A robust system of components

A very flexible separation of presentation and data access

A versatile object-oriented database independent data access framework

A consistent security scheme for both tasks and objects

A simple user and group-management system

A convenient packaging system for distributing and installing a applications
