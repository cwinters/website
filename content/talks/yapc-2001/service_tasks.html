The main service

What does the main service do?

Registers itself with persistent environment (e.g., Apache handler)

Acts as the gatekeeper: handles every request

For each request we:

create a database connection

call state preservation mechanism (web = cookies) to create session

determine what user this is

determine to what groups this user belongs

map request to top-level content handler

call top-level content handler with request

preserve state

return content
