Simple template example

So what does a template look like?

[%- comp( 'showerror', error_msg = error_msg ) -%]

<div align="center">

<table border="0" width="70%" cellpadding="5" cellspacing="0">
<tr><td align="left">
<h2 align="center">User Listing</h2>

<p>Listed below are all the users currently in the system. You can
get further detail about each one or perform other actions on them.</p>
</td></tr>
</table>

<table border="0" bgcolor="[% th.border_color %]" 
       cellspacing="0" cellpadding="2">
<tr><td>

 <table border="0" width="100%" bgcolor="[% th.bgcolor %]" 
        cellpadding="5" cellspacing="0">
[%- font_header = "color='$th.head_font_color' size='+1'" -%]
  <tr bgcolor="[% th.head_bgcolor %]" align="center">
   <td><font [% font_header %]">Username</font></td>
   <td><font [% font_header %]>Full Name</font></td>
   <td><font [% font_header %]>Title</font></td>
   <td><font [% font_header %]>E-Mail</font></td>
   <td><font [% font_header %]>Actions</font></td>
  </tr>

[% FOREACH user = user_list -%]
  [%- SET row_color = th.odd_color -%]
  [%- SET row_color = th.even_color IF loop.count mod 2 == 0 -%]
  <tr bgcolor="[% row_color %]">
    <td><font size="-1">[% user.login_name %]</font></td>
    <td><font size="-1">[% user.full_name %]</font>&nbsp;</td>
    <td><font size="-1">[% user.title %]</font>&nbsp;</td>
    <td><font size="-1">&nbsp;
[%- IF user.email -%]
        <a href="mailto:[% user.email %]">[% user.email %]</a>
[%- END -%]
    </font></td>
    <td><font size="-1">
     <a href="/User/show/?user_id=[% user.user_id %]">Detail</a><br>
     <a href="/User/remove/?user_id=[% user.user_id %]">Remove</a><br>
    </font></td>
  </tr>
[% END -%]

  <tr bgcolor="[% th.head_bgcolor %]"><td colspan="5">&nbsp;</td></tr>
  
 </table>
</td></tr>
</table>
</div>
