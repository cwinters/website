What is an Application Server?

Application server:

Not just a database, but a service (so Microsoftian...)

"n-tier" - great party buzzword!

sits between the client and the data store(s) with business rules and logic about the data

Provides development infrastructure (authentication, security, display, data access, distributed stuff)

Provides APIs for expansion and extension of services

Examples: Apache (lower-level), J2EE implementations, Kiva/iPlanet/...
