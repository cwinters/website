Start with the Metadata

  • Database-driven system, start with database metadata
  • Reading from DB every time we generate is slow
  • We want to be agile
  • So generate the metadata too
  • The real reason: MS SQL Server will never run on Linux
  • Example:
[[GU_Contract]]
PK: gasc_gucno
gasc_entno :: 12 :: NOT NULL ::  :: varchar (4)
gasc_gucno :: 4 :: NOT NULL ::  :: int identity (10)
gasc_ordno_line :: 4 :: NULL ::  :: int (10)
gasc_ref_gucno :: 12 :: NULL ::  :: varchar (20)
gasc_rscno :: 12 :: NOT NULL ::  :: varchar (35)
gasc_locno :: 12 :: NOT NULL ::  :: varchar (15)
gasc_guono :: 12 :: NOT NULL ::  :: varchar (20)
gasc_date_req :: 93 :: NULL ::  :: datetime (23)
