Gotchas: Build Process

  • Benefit: Perl runs everywhere!
  • Tight iteration cycles
  • Generation process MUST be easy to call and configure
  • Generation process MUST run quickly (relative)
  • Generation process MUST integrate into automated build
<target name="generate_beans" depends="init">
    <exec executable="perl" dir="${basedir}">
        <arg value="${generate}/create_beans.pl"/>
        <arg value="--dir=${generate}"/>
        <arg value="--bean=${bean_name}"/>
    </exec>
</target>

To call:
$ ant generate_beans
$ ant generate_beans -Dbean_name=Customer
  • You have automated builds, right?
