One-minute intro (2)

  • Create a minimal SPOPS configuration
  • Class name, table name, database connection info
  • Perl data structure:
$job = {
  job => {
     class      => 'My::Job',
     isa        => [ 'SPOPS::DBI::SQLite', 'SPOPS::DBI' ],
     rules_from => [ 'SPOPS::Tool::DBI::Datasource' ],
     field      => [ 'id', 'name', 'occupation' ],
     id_field   => 'id',
     base_table => 'job',
     dbi_config => { dsn      => 'DBI:SQLite:dbname=spops.db',
                     username => '',
                     password => '' },
  },
};
