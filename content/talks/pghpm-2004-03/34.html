Side-effect Example: Refactor for testing

  • "Every problem can be solved by adding one or more abstraction layers"
  • Refactor our application for mocks
  • First, create a wrapper for session creation and deletion
package MyWebUtils;

sub get_session {
    my ( $class, $session_id ) = @_;
    return My::SessionManager->create( $session_id );
}

sub save_session {
    my ( $class, %session ) = @_;
    My::SessionManager->save( %session );
}
