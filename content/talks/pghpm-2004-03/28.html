Side-effect Example: Intro

  • We want to test our webapp which uses sessions
  • Well-structured web apps may have something like:
package MyWebUtils;

sub get_session {
    my ( $class, $session_id ) = @_;
    my %session = ();
    tie %session, 'Apache::Session::MySQL', $session_id,
                  { DataSource => 'dbi:mysql:sessions' };
    return %session;
}

sub save_session {
    my ( $class, %session ) = @_;
    untie %session;
}
