What makes this site run

This site runs on a P-200 MMX/128 MB RAM/6.4 GB Hard drive/S3 Trio 3D 4MB. It also acts as the firewalling gateway between my network at home and the Internet.

The software it uses is:

  • Apache 1.3.14
  • Perl 5.005_03
  • mod_perl 1.24_01
  • OpenInteract 1.03
  • SPOPS 0.35
  • MySQL 3.23
  • Some custom OpenInteract packages
  • Lots of perl modules from the community

What I'm interested in doing with this site is creating a test bed. In this test bed I'll try out a number of things, including:

  • being able to change the look of the site very easily
  • allowing you to personalize how you see the information here -- via themes or what have you
  • updating much of the information on the site via web-based editing interface
  • allowing administrators to edit (almost in-place) much of the dynamic content on the site

In general, I'd like to progress to the point where Barb or another non-computer person could be given a password and edit any information on the site. Content management would be nice (particularly with something like WebDAV operating in the backend), but that might be a little ways down the road.
